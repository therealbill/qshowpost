NAME

qshowpost - A tool for displaying current Postfix queues  

SYNOPSIS

qshowpost [--help] [--meta] [--config=configfile] [interval]  

DESCRIPTION

qshowpost examines the on-disk queue directories of Postfix to determine and display current queue counts. It utilizes a configfile to tell it what instances to check and what queues to look into. It also supports interval checking to run continuously at a specificed interval. This is useful for monitoring queue flow and/or direction.

 

OPTIONS

qshowpost requires no options but accepts several.

--meta
Display application metadata
--help
Show usage
--config=FILE
Use FILE as configuration file
[number]
An optional interval to recheck and add additional results to current display
